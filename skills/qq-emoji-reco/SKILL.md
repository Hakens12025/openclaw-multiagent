# QQ表情识别

## 常见QQ表情对照表

### 微笑类
- `[微笑]` / `/wx` - 基础微笑
- `[大笑]` / `/dx` - 开心大笑
- `[偷笑]` / `/tx` - 偷偷笑
- `[憨笑]` / `/hs` - 憨厚笑

### 哭笑类
- `[哭]` - 哭泣
- `[大笑]` - 开心
- `[尴尬]` / `/gg` - 尴尬
- `[抠鼻]` / `/kb` - 抠鼻

### 表情类
- `[斜眼笑]` / `/xyx` - 斜眼笑
- `[呲牙]` / `/cy` - 呲牙
- `[调皮]` / `/tp` - 调皮
- `[流汗]` / `/lh` - 流汗

### 动作类
- `[握手]` / `/ws` - 握手
- `[玫瑰]` / `/mg` - 玫瑰花
- `[爱心]` / `/ax` - 爱心
- `[强]` - 强

### 其他常用
- `[疑问]` / `/yw` - 疑问
- `[惊讶]` / `/jy` - 惊讶
- `[撇嘴]` / `/pz` - 撇嘴
- `[白眼]` / `/by` - 白眼

## 识别方法

1. **文本模式匹配**: 搜索消息中的 `[表情名]` 格式
2. **emoji转换**: 某些QQ表情对应Unicode emoji
3. **文字描述**: 如 "微笑"、"大笑" 等文字

## 使用场景

- 分析QQ聊天记录的情绪
- 统计对话中的表情使用
- 提取消息中的表情元数据
