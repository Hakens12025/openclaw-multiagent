# 模型推荐与切换

根据用户描述的任务，推荐最合适的 AI 模型并指导切换。
当前系统默认统一使用 `ark-openai/minimax-m2.5`。只有在任务明显更适合其他模型时，才建议用户切换。

---

## AI 决策指南

### 任务类型 → 推荐模型

| 用户任务 | 推荐模型 | 原因 |
|----------|----------|------|
| 日常聊天、快速问答 | MiniMax M2.5（默认） | 最快，默认统一模型 |
| 写中文文章、翻译、中文理解 | GLM-4.7 | 中文表达稳定 |
| 代码实现、调试、重构 | Doubao Seed 2.0 Code | 代码任务更对口 |
| 长文分析、资料整理 | Kimi K2.5 | 长上下文和中文阅读更稳 |
| 复杂分析、综合推理 | DeepSeek V3.2 | 综合推理较强 |

### 意图识别

| 用户说法 | 意图 | 处置 |
|----------|------|------|
| "帮我用更好的模型" | 推荐并切换 | 询问任务类型后推荐 |
| "切换到 GLM" / "换成 GLM" | 切换特定模型 | 直接给出切换命令 |
| "哪个模型适合写作文" | 推荐咨询 | 给出推荐 + 切换命令 |
| "当前用的什么模型" | 查询状态 | 告知用 `/status` 查看 |
| "换回默认" / "用默认模型" | 切换回 MiniMax | 给出切换命令 |

### 必须追问的情况

用户说"帮我换个模型"但没说做什么任务时，先问：
> "你打算用来做什么？写文章、改代码、还是日常聊天？"

## /new 与 /model 的区别

| 命令 | 效果 |
|------|------|
| `/new <模型ID>` | 新建对话并切换，历史不带过去 |
| `/model <模型ID>` | 保持当前对话，原地切换模型 |

推荐：开始新话题用 `/new`；中途想换更合适的模型继续聊用 `/model`。

## 模型速查

| 模型 | /new 命令 | /model 命令 | 速度 | 成本 | 适合 |
|------|-----------|-------------|------|------|------|
| MiniMax M2.5 | `/new ark-openai/minimax-m2.5` | `/model ark-openai/minimax-m2.5` | 快 | 默认 | 日常聊天、通用任务 |
| GLM-4.7 | `/new ark-openai/glm-4.7` | `/model ark-openai/glm-4.7` | 快 | 默认 | 中文写作、翻译 |
| Doubao Seed 2.0 Code | `/new ark-openai/doubao-seed-2.0-code` | `/model ark-openai/doubao-seed-2.0-code` | 快 | 默认 | 代码任务 |
| DeepSeek V3.2 | `/new ark-openai/deepseek-v3.2` | `/model ark-openai/deepseek-v3.2` | 中 | 默认 | 综合推理、复杂分析 |
| Kimi K2.5 | `/new ark-openai/kimi-k2.5` | `/model ark-openai/kimi-k2.5` | 中 | 默认 | 长文整理、资料阅读 |

## 用户交互模板

### 推荐模型（带原因）

```text
根据你的任务，推荐用 {模型名}，{原因}。

发送这条命令切换：
{切换命令}
```

### 查询当前模型

```text
在对话框输入 /status 可以看到当前使用的模型。
```

### 切换成功后提示

```text
已切换到 {模型名}。{该模型的一句话特点}，继续说你的需求。
```

## 场景示例

### 场景 1：用户要写中文文章

**用户**: 帮我换个适合写文章的模型

**AI**: 写中文文章推荐 GLM-4.7，中文理解和表达都更稳。

在对话框发送：
```text
/new ark-openai/glm-4.7
```

---

### 场景 2：用户要做代码任务

**用户**: 我要改一段代码，顺便查 bug，用哪个模型好

**AI**: 代码实现和调试优先用 Doubao Seed 2.0 Code，会更贴代码场景。

在对话框发送：
```text
/new ark-openai/doubao-seed-2.0-code
```

---

### 场景 3：切换回默认

**用户**: 换回默认模型吧

```text
/model ark-openai/minimax-m2.5
```

这条命令会保留当前对话历史。
